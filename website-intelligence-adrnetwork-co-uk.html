Technical SEO Audit
Infrastructure, crawlability, and on-page technical signals — Score: 25/100
Critical Missing Meta Descriptions — All Pages
▼

0 of 7 crawled pages have a meta description

Meta descriptions are the primary copy shown under the page title in search results. Without them, Google generates its own from page content — typically poorly matched to search intent. This directly reduces organic click-through rates.

Write distinct 145–155 character meta descriptions for every key page, leading with the primary keyword and including a specific CTA verb (e.g. "Register today", "Get a quote", "Find your nearest hub").

Blog H1 reads simply "Blog" — no keyword value

The blog index page has an H1 of "Blog" — a single generic word with no keyword targeting. This page could rank for informational HGV and logistics queries but currently provides zero signal to search engines.

Change H1 to "HGV Driver & Logistics Industry Insights | ADR Network Blog" and add a 40-word intro paragraph using target keywords.

No Open Graph or Twitter Card tags detected

Without OG tags, any page shared on LinkedIn or Facebook will display an auto-selected thumbnail and truncated title. For a B2B agency sharing industry content and job listings, this significantly reduces social amplification.

Add og:title, og:description, og:image and twitter:card to all pages. Prioritise the homepage, /for-clients/, and the blog.

No schema markup of any type detected sitewide

No Organisation, LocalBusiness, JobPosting, BreadcrumbList, or Article schema found on any page crawled. This is a major missed opportunity for rich results in Google Search, particularly for job listings.

Implement at minimum: Organisation schema on homepage, JobPosting schema on each vacancy page, BreadcrumbList on interior pages, and Article schema on blog posts.

About page returns 404 — significant trust signal gap

The About page (/about-us/) returned a 404 error during the audit. An About page is a direct E-E-A-T signal and a key trust page for both search engines and potential business clients evaluating ADR Network.

Create or restore the About page with company history, team introductions, named leadership, Pertemps Group relationship, and Chartergate Certified status.

Hub Locator page uses JavaScript-rendered content — crawl risk

The recruitment hubs page uses a JavaScript-powered postcode search tool with no static list of hub locations. If Googlebot cannot render the JS, none of the location data is indexable — meaning local SEO value from hub addresses is zero.

Add a static fallback list of all UK hub names, cities, and addresses below the interactive tool. This is visible to both users and search crawlers.
